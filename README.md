## 📖 Overview
**StepSis** is a lightweight and efficient server-side plugin for **Minecraft 1.21.10** that enables players to place **shulker boxes inside other shulker boxes**.

Built with safety and performance in mind, the plugin prevents duplication glitches and handles inventory interactions cleanly without impacting server performance(hopefully lol).

## ✨ Features
- 📦 Store shulkers inside shulkers
- 🔒 Anti-dupe protection
- ⚡ Minimal performance impact

## 🚀 Installation
1. Navigate to your server's `plugins/` directory  
2. Drop the compiled `StepSis.jar` into the folder  
3. Restart your server  

## 🎮 Usage
- Open any shulker box  
- Place another shulker inside  
- Done.

## 🛠️ Developer Setup (Cloning)

### 1. Clone the Repository
```bash
git clone https://github.com/YanirValdman/StepSis.git
```

### 2. Import to IntelliJ IDEA
- Open IntelliJ IDEA → **Open Project**
- Select the cloned folder  
- Make sure **Spigot API (1.21.10)** is added as a dependency  

### 3. Build the JAR
- Go to **File → Project Structure → Artifacts**
- Add a **JAR from modules with dependencies**
- Build via **Build → Build Artifacts**
